# Restaurant Ordering System

## Description
A Java GUI-based Restaurant Ordering System developed using Object-Oriented Programming concepts. The system allows users to select different food categories, enter item details and quantity, calculate discounts, and generate the final bill.

## Features
- GUI-based user interface using Java Swing
- Inheritance using a FoodItem base class
- Derived classes:
  - Beverage
  - FastFood
  - TraditionalFood
- Method overriding through calculateBill()
- Category-specific discount calculation
- Automatic final bill generation
- Display of item details, discount, and payable amount

## OOP Concepts Used
- Classes and Objects
- Inheritance
- Method Overriding (Runtime Polymorphism)
- Encapsulation
- GUI Development with Swing

## How It Works
1. Enter the item name.
2. Enter the quantity.
3. Select the food category.
4. The system calculates the bill according to the selected category.
5. Applicable discounts are applied.
6. Final price and discount details are displayed.

## Technologies Used
- Java
- Java Swing
- Object-Oriented Programming (OOP)

- Building GUI applications with Java Swing
- Applying OOP concepts in a real-world scenario
